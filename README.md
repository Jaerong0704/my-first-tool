# AI Tools Hub 🚀

2026년 최고의 AI 도구 200개를 한 곳에서!

## 📌 소개

AI Tools Hub는 전 세계의 다양한 AI 도구들을 카테고리별로 정리하고, 각 도구의 상세 정보를 제공하는 웹사이트입니다.

## ✨ 주요 기능

- 🔍 **실시간 검색** - 도구 이름, 설명, 태그로 즉시 검색
- 🎯 **카테고리 필터** - 글쓰기, 이미지, 영상, 코딩 등 8개 카테고리
- 💰 **가격 필터** - 무료/유료 도구 구분
- 📱 **반응형 디자인** - 모바일, 태블릿, PC 모두 지원
- 🎨 **상세 정보 모달** - 각 도구의 장단점, 가격, 추천 대상 확인
- 🌈 **모던한 UI** - 화려한 그라디언트와 부드러운 애니메이션

## 🎯 대상 사용자

- AI 도구를 찾고 있는 개인/기업
- 새로운 AI 도구를 발견하고 싶은 크리에이터
- AI 도구 비교가 필요한 의사결정자
- 최신 AI 트렌드를 따라가고 싶은 얼리어답터

## 💡 사용 방법

1. **검색** - 상단 검색창에 키워드 입력
2. **필터** - 카테고리와 가격으로 필터링
3. **클릭** - 관심있는 도구 카드 클릭
4. **확인** - 모달창에서 상세 정보 확인
5. **사용** - "사용하기" 버튼으로 도구 사이트 방문
